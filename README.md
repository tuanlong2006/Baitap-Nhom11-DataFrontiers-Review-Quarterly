# Baitap-Nhom11-DataFrontiersReviewQuarterly
Bài tập lớn môn Cơ sở dữ liệu -DataFrontiers  Review  Quarterly 
DataFrontiers Journal Management System


Đề tài xây dựng cơ sở dữ liệu cho hệ thống quản lý tạp chí khoa học DataFrontiers, hỗ trợ quản lý:

Tác giả (Author)
Bài báo (Manuscript)
Chuyên gia phản biện (Reviewer)
Lĩnh vực chuyên môn (Expertise Area)
Phân công phản biện (Review Assignment)
Số phát hành tạp chí (Issue)

Hệ thống được thiết kế theo mô hình ERD và triển khai trên MySQL.
